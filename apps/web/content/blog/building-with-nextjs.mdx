Next.js is a full-stack React framework that makes building fast web applications straightforward.

## Why Next.js?

A few reasons I chose Next.js for this site:

1. **App Router** — nested layouts, server components, and streaming out of the box
2. **MDX support** — write blog posts in markdown with React components
3. **Tailwind CSS** — utility-first styling that scales

## Setting Up MDX

With `@next/mdx`, adding MDX support is just a few lines of config:

```ts
import createMDX from "@next/mdx";

const withMDX = createMDX({
  options: {
    remarkPlugins: ["remark-frontmatter"],
  },
});

export default withMDX(nextConfig);
```

That's it — now `.mdx` files are first-class pages and importable modules.
